# Edge Cases — AI Automation Agency

> We solve the automation problems templates can't.

Premium agency website for **Edge Cases**, built with React, Vite, Tailwind CSS v4, and Framer Motion.

## Tech Stack

- **React 19** + TypeScript
- **Vite 6** — dev server & bundler
- **Tailwind CSS v4** — utility-first styling
- **Framer Motion** — page transitions & micro-animations
- **Lucide React** — premium icon set

## Run Locally

```bash
npm install
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
├── App.tsx                 # Router & page transitions
├── main.tsx                # React entry point
├── index.css               # Design tokens & global styles
├── types.ts                # Shared TypeScript types
├── components/
│   ├── Header.tsx          # Sticky nav with animated pill
│   ├── Footer.tsx          # Dark footer with links
│   └── FloatingWhatsApp.tsx # Fixed WhatsApp CTA
├── pages/
│   ├── Home.tsx            # Hero, trust, featured solutions
│   ├── Services.tsx        # Service cards, timeline, retainer tiers
│   ├── About.tsx           # Story, founders, approach, tools
│   ├── CaseStudies.tsx     # Case study grid & detail view
│   └── Contact.tsx         # Contact form & info
└── data/
    ├── casestudies.ts      # Case study content
    └── tools.ts            # Tool stack data
```
